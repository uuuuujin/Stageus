## 부족한 점 
* 회원가입
1. 비밀번호에 영문자 + 숫자 + 특수문자 형태로 안 만듦 (현재 규칙 없음)
2. 전화번호에 아무 숫자나 입력해도 회원가입 가능함  
<br>

* 일정 추가 or 수정
1. 날짜에 숫자 1~5자리로 입력하면 에러남 
2. 날짜 yymmdd 형식으로 입력하지 않아도 저장됨 (예. 111111으로 해도 저장)
